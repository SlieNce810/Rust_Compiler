## 10. 当前建议

如果你还没定下来，建议先按这套最小方案推进：

- 芯片：先固定一块主控板
- 下载链路：先 UART
- 启动方式：bootloader 上电等待短时间，再跳应用
- 更新对象：先下发完整字节码包，不先做完整固件 OTA
- 校验：先做 `magic + version + length + crc32`
- 回滚：第一版先不做 A/B，只保留 bootloader + 单应用区
